# symbiot-ui

A Web UI for Symbiot

# Try it

## Prerequisites

* Node.js
* Bower

## Then it's a matter of:

* `> npm install`
* `> bower install`
* `> gulp`
* Open `index.html`

You can run `gulp watch` if you are editing the Javascript or Sass sources, which will automatically re-process what is needed.
